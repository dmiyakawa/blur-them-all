# Blur Them All

ブラウザ上で動作する、モザイク・ブラー処理に特化したシンプルな画像編集ツールです。

## 特徴

- **完全クライアントサイド**: 画像データは外部に送信されず、すべてブラウザ内で処理されます
- **シンプルで直感的**: 必要な機能だけに絞った使いやすいインターフェース
- **インストール不要**: ブラウザだけで動作します
- **プライバシー重視**: 画像はあなたのデバイス内でのみ処理されます

## 主な機能

### 画像編集
- **モザイク処理**: 矩形範囲を選択してピクセレーション処理
- **ブラー処理**: 矩形範囲を選択してガウシアンブラー適用
- **ペン注釈**: 色と太さを変えられるペンで自由に描画
- **矢印配置**: 注目させたい箇所に矢印を配置

### その他
- **Undo/Redo**: 操作の取り消し・やり直し（最大30ステップ）
- **画像エクスポート**: PNG形式でダウンロード
- **ドラッグ&ドロップ**: 画像を簡単に読み込み

## 使い方

### 1. 画像を読み込む

- 画像をドラッグ&ドロップ、または「画像を選択」ボタンをクリック
- 対応形式: PNG, JPEG, GIF, WebP（最大10MB）

### 2. ツールを選択

左側のツールバーから使用したいツールを選択します：

- **選択**: デフォルトのツール
- **ペン**: 自由に描画
- **矢印**: 矢印を配置
- **ブラー**: 選択範囲をぼかし
- **モザイク**: 選択範囲にモザイクをかける

### 3. 編集する

#### ペンで描画
1. ペンツールを選択
2. 下部のプロパティパネルで色と太さを調整
3. Canvas上でドラッグして描画

#### 矢印を配置
1. 矢印ツールを選択
2. 下部のプロパティパネルで色、太さ、矢印サイズを調整
3. Canvas上でドラッグして矢印を作成

#### ブラー・モザイクを適用
1. ブラーまたはモザイクツールを選択
2. 下部のプロパティパネルで強度を調整
3. Canvas上で矩形範囲をドラッグして選択
4. 選択を確定すると自動的に適用されます

### 4. エクスポート

右上の「画像をエクスポート」ボタンをクリックしてPNG形式でダウンロードします。

## 開発

### 必要な環境

- Node.js 18以上
- npm または yarn

### セットアップ

```bash
# 依存パッケージのインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

### プロジェクト構成

```
blur-them-all/
├── src/
│   ├── components/       # Vueコンポーネント
│   ├── composables/      # Vue Composition API
│   ├── services/         # コアサービス
│   ├── types/            # TypeScript型定義
│   ├── utils/            # ユーティリティ関数
│   ├── constants/        # 定数定義
│   ├── assets/           # アセット（CSS、アイコン）
│   ├── App.vue           # ルートコンポーネント
│   └── main.ts           # エントリーポイント
├── apache/               # Apache設定ファイル
├── public/               # 静的ファイル
├── .ai/                  # AI関連ドキュメント
├── AGENTS.md             # AI開発者向けガイド
└── README.md             # このファイル
```

詳細な設計については `.ai/design.md` を参照してください。

## デプロイ

### Apache 2.4 へのデプロイ

1. プロジェクトをビルド:
   ```bash
   npm run build
   ```

2. `dist/` ディレクトリの内容をサーバーにアップロード

3. `.htaccess` ファイルをコピー:
   ```bash
   cp apache/.htaccess dist/
   ```

4. Apache の設定例を参照:
   ```bash
   cat apache/httpd.conf.example
   ```

5. 必要なApacheモジュールを有効化:
   ```bash
   # Ubuntu/Debian
   sudo a2enmod rewrite deflate expires headers
   sudo systemctl restart apache2
   ```

### 静的ホスティングサービス

以下のサービスでも簡単にデプロイできます：

- **Vercel**: プロジェクトルートで `vercel` コマンドを実行
- **Netlify**: `dist/` ディレクトリをドラッグ&ドロップ
- **GitHub Pages**: GitHub Actionsを使用して自動デプロイ
